## Functional Test란?
Unit Test와 달리 실제 환경에서 전체 기능을 검증하는 테스트
- 실제 OpenStack 환경에서 실행
- 실제 HTTP 요청을 서버에 전송
- 실제 데이터베이스에 리소스 생성/수정/삭제
- End-to-End 시나리오 검증
